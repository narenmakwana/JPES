# Allstate-JPES
### Prerequisites
Node.js, NPM, Bower

##Client

### Install

```
cd client
[sudo] npm install
bower install
```

### Start development server

For the client:
```
cd client
gulp serve
```

### Working with production code
Build:
```
cd client
gulp build
```

Run:
```
cd client
gulp serve:dist
```
##Server

###Install
```
cd server
[sudo] npm install
```

### Start development server
```
cd server
npm start
```
Server listens at http://localhost:3003/
